CentOS7_LNMP(第二篇)_MariaDB10.0.21编译安装

发表于 2015-10-23   |   分类于 CentOS ， LNMP ， MariaDB   |     |   阅读量 次

CentOS7+Nginx1.95+mariadb10.0.21+php5.6.14

1.安装MariaDB和一些必要的组件
[root@XiaoFeng ~]# yum install -y libxml2 libxml2-devel bison bison-devel
2.编译安装Cmake

3.安装MariaDB
[root@XiaoFeng lnmp]# tar zxvf mariadb-10.0.21.tar.gz

#####配置用户和用户组/数据目录：

创建mysql用户,不允许mysql用户直接登录系统
[root@XiaoFeng mariadb-10.0.21]#useradd -s /sbin/nologin mysql
创建MariaDB数据库存放目录
[root@XiaoFeng mariadb-10.0.21]#mkdir -p /data/mysql
设置MariaDB数据库目录权限
[root@XiaoFeng mariadb-10.0.21]#chown -R mysql:mysql /data/mysql
创建MariaDB安装目录
[root@XiaoFeng mariadb-10.0.21]#mkdir -p /usr/local/mysql
设置目录权限
[root@localhost mariadb-10.0.21]# chown -R mysql:mysql /usr/local/mysql

#####开始编译安装

拷贝配置文件（注意：如果/etc目录下面默认有一个my.cnf，直接覆盖即可）
[root@XiaoFeng mariadb-10.0.21]#cd /usr/local/mysql
[root@XiaoFeng mysql]#cp ./support-files/my-huge.cnf /etc/my.cnf
编辑配置文件,在 [mysqld] 部分增加
datadir = /data/mysql #添加MariaDB数据库路径
[root@XiaoFeng mysql]# vim /etc/my.cnf
初始化数据库
[root@XiaoFeng mysql]#./scripts/mysql_install_db --user=mysql --datadir=/data/mysql
复制启动文件到开机自动启动
[root@XiaoFeng mysql]#cp ./support-files/mysql.server /etc/rc.d/init.d/mysql
增加执行权限
[root@XiaoFeng mysql]# chmod 755 /etc/init.d/mysql
添加到启动服务里
[root@XiaoFeng mysql]# chkconfig --add mysql
加入开机启动
[root@XiaoFeng mysql]# chkconfig mysql on
查看开机启动
[root@XiaoFeng mysql]# chkconfig --list mysql
mysql 0:关闭 1:关闭 2:启用 3:启用 4:启用 5:启用 6:关闭
编辑启动文件加入下面两行
[root@XiaoFeng mysql]# vim /etc/rc.d/init.d/mysql
basedir = /usr/local/mysql #MariaDB程序安装路径
datadir = /data/mysql #MariaDB数据库存放目录
启动服务
[root@XiaoFeng mysql]# service mysql start
把MariaDB服务加入系统环境变量：在最后添加下面这一行
export PATH=$PATH:/usr/local/mysql/bin
[root@XiaoFeng mysql]# vim /etc/profile
下面这两行把MariaDB的库文件链接到系统默认的位置，这样你在编译类似PHP等软件时可以不用指定MariaDB的库文件地址。
[root@XiaoFeng mysql]# ln -s /usr/local/mysql/lib/mysql /usr/lib/mysql
[root@XiaoFeng mysql]#ln -s /usr/local/mysql/include/mysql /usr/include/mysql
需要重启系统，等待系统重新启动之后继续在终端命令行下面操作
[root@XiaoFeng mysql]# shutdown -r now
设置MariaDB数据库root账号密码
[root@XiaoFeng mysql]# mysql_secure_installation
根据提示按Y 回车输入2次密码
或者直接修改密码/usr/local/mysql/bin/mysqladmin -u root -p password “123456” #修改密码
重启数据库
[root@XiaoFeng mysql]# service mysql restart
输入上面设置的root密码登录到mariadb控制台
[root@XiaoFeng mysql]# mysql -u root -p
