centos7_lnmp_nginx1.95编译安装

发表于 2015-10-23   |   分类于 CentOS ， Nginx   |  

CentOS7+Nginx1.95+mariadb10.0.21+php5.6.14

1.查看系统版本
root@XiaoFeng html]#cat /etc/centos-release CentOS Linux release 7.1.1503
2.安装一些必要的组件
[root@XiaoFeng /]# yum -y install libjpeg-devel libpng-devel libtiff-devel fontconfig-devel freetype-devel libXpm-devel gettext-devel openssl-devel libtool-ltdl-devel gcc *c++* ncursesncurses-devel

3.添加Nginx开机启动
启动脚本
vim /usr/lib/systemd/system/nginx.service

#输入下面内容，并保存

#保存为nginx.service
4.修改权限
[root@XiaoFeng /]#chmod +x /usr/lib/systemd/system/nginx.service
5.开机启动
[root@XiaoFeng /]#systemctl enable nginx.service

#现在可以使用下面的指令来控制nginx啦

6.查看日志
[root@XiaoFeng /]#journalctl -f -u nginx.service

#注意上面的几个路径,下面这几个路径是你的nginx安装的目录，务必不要弄错。

7.启动服务
[root@XiaoFeng ~]# systemctl start nginx.service
